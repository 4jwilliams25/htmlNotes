The HTML Page

The header element is used to hold header element

Often holds Logo, title, navigation, etc

This is the main Element

This wraps around the main content of the page

This is a section element

These are used to notate a section of content. Let's start another thing. Usually has a headline.

This is an aside element

Used to mark something that isn't main content

This is an article element

Article element is wrapped around any instance of an article large or small. Just means 'this' by itself is a unit of content

Footer is used to hold a footer element

The footer doesn't necessarily need to be at the bottom. It is the semantic meaning that matters more than the position
